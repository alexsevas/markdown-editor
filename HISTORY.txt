v0.0.3
--------
============================================
ОПИСАНИЕ ВСЕХ ФУНКЦИЙ И ВНЕСЕННЫХ ИЗМЕНЕНИЙ
============================================

1. ИСПРАВЛЕНИЕ ПРОБЛЕМЫ С КОДИРОВКОЙ:
   - Добавлено меню "Encoding" с выбором популярных кодировок
   - Реализована ручная смена кодировки через меню
   - При переключении кодировки файл автоматически перечитывается
   - Сохраняется позиция курсора и прокрутки при смене кодировки
   - Добавлена статусная метка, отображающая текущую кодировку
   - Поддержка основных кодировок для кириллицы: UTF-8, Windows-1251, KOI8-R и др.
   - Сохранение выбранной кодировки в настройках приложения

2. РЕАЛИЗАЦИЯ ТЕМНОЙ ТЕМЫ:
   - Добавлен переключатель "Dark Mode" в меню View (Ctrl+D)
   - Полная настройка цветовой схемы для редактора и превью
   - Автоматическое обновление подсветки синтаксиса при переключении темы
   - Сохранение состояния темы между сессиями через QSettings
   - Специальные цвета для темной темы во всех элементах интерфейса
   - Темные стили для превью Markdown (заголовки, код, цитаты и т.д.)

3. УЛУЧШЕНИЕ ПОДСВЕТКИ СИНТАКСИСА:
   - Полная реализация подсветки кода
   - Добавлено расширение codehilite для правильной обработки блоков кода
   - Реализована детальная цветовая схема для всех элементов кода:
     * Комментарии, ключевые слова, строки, числа и т.д.
     * Поддержка различных языков программирования
   - Автоматическое применение темных стилей в темной теме
   - Улучшенная подсветка для Markdown-синтаксиса (заголовки, жирный, курсив и т.д.)

4. ИСПРАВЛЕНИЕ ОШИБОК И ПРЕДУПРЕЖДЕНИЙ:
   - Добавлено подавление предупреждений DeprecationWarning в начале скрипта
   - Исправлены ошибки с экранированием фигурных скобок в HTML-шаблоне
   - Улучшена обработка ошибок при открытии и сохранении файлов
   - Добавлена обработка ошибок кодировки с пользовательскими сообщениями
   - Исправлен импорт QSettings (теперь из PyQt5.QtCore)

5. ДОПОЛНИТЕЛЬНЫЕ УЛУЧШЕНИЯ:
   - Сохранение настроек приложения между запусками (тема, кодировка)
   - Улучшенный интерфейс с разделением на панели редактора и превью
   - Поддержка LaTeX-формул через MathJax в превью
   - Все горячие клавиши работают как в стандартных текстовых редакторах
   - Возможность скрыть панель превью для большего рабочего пространства
   - Статусная строка с информацией о текущих операциях

6. ТЕХНИЧЕСКИЕ УЛУЧШЕНИЯ:
   - Использование QSettings для хранения пользовательских настроек
   - Добавлен debounce-таймер для оптимизации обновления превью
   - Улучшенная обработка ошибок с пользовательскими сообщениями
   - Поддержка всех основных операций с файлами (новый, открыть, сохранить)
   - Автоматическое определение кодировки при открытии файлов


v 0.0.2
--------
Что изменилось:
- Улучшенное определение кодировки:
- Сначала проверяем наличие BOM (Byte Order Mark) для UTF-8
- Если уверенность chardet ниже 70% или определена ASCII, но файл содержит не-ASCII символы
- Автоматически пробуем распространённые кодировки для кириллицы:
   - cp1251 (Windows-1251) - основная кодировка для русского в Windows
   - koi8-r - альтернативная кодировка для кириллицы
   - utf-8 - как fallback
   - latin1 - для общего случая

Безопасное чтение:
- Если ни одна кодировка не подошла, используем errors='replace' чтобы избежать падения программы
- Проверяем первые 1000 байт на наличие не-ASCII символов
